# AWS Boto
Boto3 Client(low-level API) vs Boto3 resource(higher-level API)

## Category

### List all S3 Bucket

* Boto3 client
```
python3 s3_client.py
```

* Boto3 resource
```
python3 s3_resource.py
```

### List all EC2 Instance

* Boto3 client 
```
python3 ec2_client.py
```

* Boto3 resource 
```
python3 ec2_resource.py
```
